Sim2Editor makes use of Sim2Editor-Core.
You can find a list of features of Sim2Editor below.

Game Boy Advance Features
General
  • Edit Sim Name
  • Edit Simoleons
  • Edit Ratings
  • Edit Time
  • Edit Collectables Amount and Sell price
  • Edit Skillpoints
Cast Member
  • Edit Friendly Interaction Level
  • Edit Romance Interaction Level
  • Edit Intimidate Interaction Level
  • Edit Feeling
  • Edit Feeling Effect Hours
  • Edit Phone Registered State
  • Edit Secret Unlocked State
Social Moves
  • Edit State
  • Edit Level
  • Edit Blocked Hours
Minigames
  • Edit Played Today State
  • Edit Level
Plot Twists
  • Edit Unlocked State
Item Packages
  • Edit Pawnshop Items
  • Edit Saloon Items
  • Edit Skillbook Seller Items
  • Edit Mailbox Items
  • Edit Inventory Items
Nintendo DS Features
General
  • Edit Sim Name
  • Edit Simoleons
  • Edot Collectables Amount
  • Edit Skillpoints
